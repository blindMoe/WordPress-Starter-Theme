This mu-plugins directory goes inside of wp-content and any plugins you put in it, are automatically included and enabled by default. "mu" stands for "must use".

Note in the comments: It only works for single file plugins, not so much for any requiring a folder hierarchy. Comments on the blog post below are a good thing to read.

More details at this url: http://justintadlock.com/archives/2011/02/02/creating-a-custom-functions-plugin-for-end-users
